# CSV Cleaner Utility

A simple Python command-line tool to clean CSV files.

## Features

- Removes duplicate rows
- Fills missing values
- Handles basic errors
- Saves cleaned CSV file

## Installation

Make sure you have Python installed and a virtual environment created with python -m venv venv.
Then, install the required dependencies:

``` bash
    pip install pandas
```

## Usage

python cleaner.py input.csv output.csv

Example:

python cleaner.py sample.csv cleaned_sample.csv
